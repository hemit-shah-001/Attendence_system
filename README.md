# Attendence_system
## Tech used: HTML, CSS, JAVASCRIUPT, BOOTSTRAP, PHP, SQL
## Steps to run:
Login using the teachers id and password
Select the division
Select 2hr, 1hr or abasent or the student
You can add or delete student
